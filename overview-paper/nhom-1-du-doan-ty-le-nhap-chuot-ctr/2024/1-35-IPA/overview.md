# Review Paper: Towards Unifying Feature Interaction Models for CTR Prediction (IPA Framework)

**ArXiv ID:** [2411.12441](https://arxiv.org/abs/2411.12441)
**Năm:** 2024
**Nhóm:** 1 - Dự đoán tỷ lệ nhấp chuột (CTR) — Kiến trúc mô hình

---

## 1. Paper này đang nghiên cứu gì?

Hệ thống hóa các mô hình tương tác đặc trưng dưới framework thống nhất:
- Các mô hình tương tác đặc trưng đa dạng nhưng không có khung thống nhất

## 2. Phương pháp sử dụng

- IPA framework: Interaction Function + Layer Pooling + Layer Aggregator
- Dimensional collapse analysis
- Giới thiệu mô hình mới PFL

## 3. Thành tựu đạt được

- Framework thống nhất phân loại hầu hết mô hình CTR hiện tại
- PFL: hiệu suất cạnh tranh với SOTA
- Triển khai sản xuất tại Tencent
- Cải thiện GMV trong online A/B testing

## 4. Hạn chế

- Phân tích dimensional collapse chưa chi tiết
- Scope chủ yếu trên CTR advertising
