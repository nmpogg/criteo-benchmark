# Review Paper: xDeepInt: a hybrid architecture for modeling vector-wise and bit-wise feature interactions

**ArXiv ID:** [2301.01089](https://arxiv.org/abs/2301.01089)
**Năm:** 2023
**Nhóm:** 1 - Dự đoán tỷ lệ nhấp chuột (CTR) — Kiến trúc mô hình

---

## 1. Paper này đang nghiên cứu gì?

Học tương tác vectơ chiều bậc cao, cân bằng vector-wise và bit-wise

## 2. Phương pháp sử dụng

Polynomial Interaction Network (PIN) + crossing subspace + chiến lược tối ưu tùy chỉnh

## 3. Thành tựu đạt được

PIN mới cho phép học đồng thời hai loại tương tác, code TensorFlow mã nguồn mở, hiệu năng cạnh tranh trên 3 dataset

## 4. Hạn chế

Không rõ hạn chế, tập trung tương tác bậc giới hạn
