# Review Paper: PEEL: Personalized Elastic Embedding Learning for On-Device Recommendation

**ArXiv ID:** [2306.10532](https://arxiv.org/abs/2306.10532)
**Năm:** 2023
**Nhóm:** 5 - Nén Embedding & Nén mô hình

---

## 1. Paper này đang nghiên cứu gì?

Recommendation trên thiết bị tài nguyên hạn chế, device và user heterogeneity

## 2. Phương pháp sử dụng

PEEL framework: pretrain global → cluster users → refine group-specific, Personalized Elastic Embeddings, diversity regularizer

## 3. Thành tựu đạt được

Embeddings compact thích ứng memory budgets, once-for-all training, superior performance trên 2 datasets

## 4. Hạn chế

Chỉ đánh giá trên 2 datasets, overhead group-wise refinement chưa rõ
