# Review Paper: CERP: Learning Compact Compositional Embeddings via Regularized Pruning for Recommendation

**ArXiv ID:** [2309.03518](https://arxiv.org/abs/2309.03518)
**Năm:** 2023
**Nhóm:** 5 - Nén Embedding & Nén mô hình

---

## 1. Paper này đang nghiên cứu gì?

Nén bảng embedding, các phương pháp lightweight gây suy giảm accuracy dưới ràng buộc bộ nhớ chặt

## 2. Phương pháp sử dụng

CERP: hai embedding bổ sung từ bảng meta nhỏ hơn, learnable element-wise thresholds, regularization mới. ICDM'23

## 3. Thành tựu đạt được

Model-agnostic, hiệu suất vượt baselines, code công khai GitHub

## 4. Hạn chế

Không nêu rõ trường hợp thất bại, vẫn có đánh đổi memory-accuracy
