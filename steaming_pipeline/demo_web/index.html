Demo động cho pipeline CTR Streaming

Upload một file Parquet, FastAPI lưu vào `data/uploads`, tạo JSON event theo số lượng bạn chọn và gửi vào Kafka. Spark realtime job là nơi đọc Kafka, xử lý feature, gọi model, ghi Data Lake/PostgreSQL. Web này theo dõi trạng thái gửi và output trong PostgreSQL.
